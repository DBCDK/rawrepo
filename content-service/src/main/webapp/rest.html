Json service information

Post content: 
{
  --optional--
  "authentication": {
    "user": "",
    "group": "",
    "password": ""
  },
  "records": {
    "record": [
      {
        "bibliographicRecordId": "",
        "agencyId": 0,
        "mode": "RAW|MERGED|COLLECTION",
        "allowDeleted":false,
        "includeAgencyPrivate":false
      },...
    ]
  }
}
to .../RawRepoService/fetch, and your "Accept" header should be one of
  • application/json
  • application/xml
  • text/xml
And your result should given. Remember to base64 decode the data element.
